Foodwall 1.0 (March 2013)


The goal of this version was to create a proof of concept with the following features:

  1. Ability to search restaurants and locations using autocompletion
  2. Ability to see food items using infinite scroll similar to Pinterest
  3. Ability to detailed nutritional information
  4. Ability to register and logout a user
  5. Ability to save items to a basket
  6. Ability to view the webpage both mobile and web browsers
